# png-to-jpg README

pngファイルをjpgファイルに変換し、参照している箇所も`xxx.png`から`xxx.jpg`に変換します。

## Features

`Shift + Ctrl (Cmd) + P` でコマンドパレットを開いて `png-to-jpg.convert` を実行してください。

## Extension Settings
以下の設定が必要です。
* "png-to-jpg.images" : "画像が置かれているディレクトリ"
* "png-to-jpg.contents" : "zenn等のコンテンツが置かれているディレクトリ"
